# The-Look-eCommerce-Sales-Customer-Analysis-Dashboard
🚀 Excited to share my first end-to-end Data Analytics project!

I'm happy to share my first complete Data Analytics project, where I worked on The Look eCommerce Dataset from start to finish.

This project gave me the opportunity to experience the complete data analysis workflow, starting from raw data and ending with an interactive dashboard that provides actionable business insights.

🔹 Project Workflow:
Data Cleaning using Power Query
Data Modeling using Power Pivot
Creating relationships between tables
Writing DAX Measures to calculate business KPIs
Building Pivot Tables and Pivot Charts
Designing an interactive dashboard in Excel.

📊 Dashboard Includes:
-Executive Overview
-Customer & Marketing Analysis
-Inventory Analysis
-Forecast Analysis.

📌 Key KPIs:
-Total Customers
-Repeat Customers & Repeat Customer Rate
-Total Orders
-Inventory Status
-Department Performance
-Distribution Center Analysis
-Traffic Source Analysis
-Browser Analysis
-Customer Demographics

💡 Through this project, I improved my skills in:
✔ Data Cleaning
✔ Data Modeling
✔ DAX Measures
✔ Data Visualization
✔ Dashboard Design
✔ Turning raw data into meaningful business insights

This project was a valuable learning experience that strengthened both my technical skills and my analytical thinking. I'm excited to continue learning, building more projects, and growing as a Data Analyst.


I would love to hear your feedback and suggestions!
